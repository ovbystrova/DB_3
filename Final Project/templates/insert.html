Добавить информацию


Вы можете добавить информацию о данных видео, о жестикуляции, о мимике и о тексте.
Если хотите добавить данные о видео в базу данных - введите сначала "insert_meta" затем (без пробелов перед числами!) введите необходимые параметры, разделяя их точкой с запятой. А именно: id_video;name;date_of_recording;location;author;lang
Если хотите добавить определенную жестикуляцию в базу данных - введите "insert_zhest" затем (без пробелов перед числами!) введите необходимые параметры, разделяя их точкой с запятой. А именно: id_video;time;zhesticulation
Если хотите добавить определенную мимку в базу данных - введите "insert_mimic" затем (без пробелов перед числами!) введите необходимые параметры, разделяя их точкой с запятой. А именно: item_id;customer_id;date
Если хотите добавить текст видео в базу данных - введите "insert_text" затем (без пробелов перед числами!) ведите необходимые параметры, разделяя их точкой с запятой. А именно: id_video;time;text
