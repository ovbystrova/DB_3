Добавить информацию


Вы можете добавить информацию о товаре, о покупателе, о покупке и о жалобе.
Если хотите добавить товар в базу данных - введите сначала "insert_item" затем (без пробелов перед числами!) введите необходимые параметры, разделяя их точкой с запятой. А именно: name;description;price
Если хотите добавить покупателя в базу данных - введите "insert_customer" затем (без пробелов перед числами!) введите необходимые параметры, разделяя их точкой с запятой. А именно: name;phone;email
Если хотите добавить покупку в базу данных - введите "insert_purchase" затем (без пробелов перед числами!) введите необходимые параметры, разделяя их точкой с запятой. А именно: item_id;customer_id;date
Если хотите добавить жалобу в базу данных - введите "insert_complaint" затем (без пробелов перед числами!) ведите необходимые параметры, разделяя их точкой с запятой. А именно: text(тут пробелы можно);item_id,purchase_id
